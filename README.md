# GlagolT

Адаптивный учебный квиз по временам английских глаголов.

## Что уже работает

- 10 вопросов с вариантами ответа;
- мгновенная проверка и объяснение;
- индикатор прогресса и подсчёт баллов;
- итоговый экран;
- повторное прохождение;
- удобный интерфейс для телефона и компьютера.

## Запуск

Откройте `index.html` в браузере. Дополнительная установка не требуется.

## Файлы

- `index.html` — структура приложения;
- `styles.css` — оформление и адаптивность;
- `app.js` — вопросы и логика квиза.
